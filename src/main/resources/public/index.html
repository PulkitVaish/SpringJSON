Get started

The SpringJSON API provides programmatic access to random data like comments and reviews.

It combines data from different popular APIs and packs them into desirable formats while providing competitive response time.

get comments

To get random comments you need to make a GET call to the following url :
https://spring-json.herokuapp.com/comments


QUERY PARAMETERS

Field Type Description
num Number A number in the range (1-100) 
fetch('https://spring-json.herokuapp.com/comments?num=10')
.then(response => response.json())
.then(json => console.log(json))
{ }
